https://micropython.org/download/ESP32_GENERIC/

Driver USB ESP32: https://www.silabs.com/developer-tools/usb-to-uart-bridge-vcp-drivers?tab=downloads

passo a passo:

> baixar bin
> abrir cmd na pasta que baixou
> no cmd: pip install esptool
> conectar a esp32 e identificar a porta COM
> no cmd: esptool --port COMX erase_flash
> pressionar botão boot do esp32 por 5 segundos
> no cmd: esptool --port PORTNAME --baud 460800 write_flash 0x1000 ESP32_BOARD_NAME-DATE-VERSION.bin
> abrir thonny python IDE
> ferramentas -> opções do thonny -> interpretador --> micropython (ESP32) -> selecionar porta da ESP32 --> ok
> vai aparecer a versão do shell do thonny "MicroPython v1.24.1 on 2024-11-29; Generic ESP32 module with ESP32"
> digitar o código abaixo:

from machine import Pin
from time import sleep

led = Pin(2, Pin.OUT)

while True:
    led.value(1)
    sleep(0.5)
    led.value(0)
    sleep(0.5)

> executar programa (F5)

> vai aparecer no shell do thonny: "MPY: soft reboot"

> voce pode salvar o arquivo "main.py" dentro do esp32 em Salvar como... ou Na aba View -> Files
