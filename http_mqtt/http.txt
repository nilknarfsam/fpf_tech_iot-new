4 MÉTODOS DE REQUISIÇÃO
GET --> bate num endpoint, retorna o valor / retorno rápido de informações (consulta?)
POST --> salva o que foi enviado ao payload  / device salva algo
DELETE --> device apagar algo de dentro do server
PUT --> editar dados 

Toda conexão HTTP (geralmente) acontece via endpoint
